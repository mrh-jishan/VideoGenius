# **App Name**: VideoGenius

## Core Features:

- Prompt Input: Allow users to input a text prompt for video generation.
- Scene Planning: Utilize Google Gemini to intelligently break down the prompt into optimized video scenes.
- Scene Editor: Allow users to manually edit the scenes proposed by Gemini: modify narration, change the prompt for the scene's images, set up transition audio.
- Keyword Modification: Enable modification of AI-generated keywords, and triggering calls to the Gemini API to find alternative words.
- Asset Selection: Provide a UI to manually select images and video clips (from Pixabay) associated with each scene.
- Narration Preview: Allow users to preview narration generated for each scene.
- JSON Output Generation: Generate a complete JSON output containing all the video specifications, formatted for the backend system.

## Style Guidelines:

- Primary color: Deep purple (#673AB7) to convey creativity and intelligence.
- Background color: Light gray (#ECEFF1) to provide a clean, modern backdrop.
- Accent color: Teal (#009688) for interactive elements and highlights.
- Body and headline font: 'Inter', a grotesque-style sans-serif, for a modern, neutral look.
- Implement a clear workflow layout, guiding the user through each step of video creation (prompt input, scene planning, asset selection, etc.).
- Use clean, modern icons to represent different actions and content types.
- Employ subtle transitions and animations to enhance user experience, like fading effects on the titles.
